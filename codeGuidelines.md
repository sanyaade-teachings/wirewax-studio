#WireWAX Studio Coding Guidelines 
**_Disobey at your peril_**

##CoffeeScript

##HTML

##File System

- All file names should be camelCased:


    `myFileName.less`
    
    `myController.coffee`

##CSS

###Selectors

- IDs should be camelCase:

    
   `#wwMyId`


- Classes should be hyphenated:

    
    `.ww-class-name`


- All selectors must be prefixed with ww:

    
    `.ww-class-name`
    
    `#wwClassName`
